# ai-study-assistant
一款面向大学生的 AI 学习助手，基于 Next.js + FastAPI 构建，支持 AI 智能问答、PDF/PPT/TXT 文件上传与自动总结、学习计划生成、复习提纲整理。采用 DeepSeek API 作为 AI 引擎，支持深色模式与响应式布局，代码简洁、注释详尽，适合初学者学习参考。An AI learning assistant for college students, featuring AI chat, file summarization, study plan generation, and dark mode. Built with Next.js + FastAPI + DeepSeek API.
